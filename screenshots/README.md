## Capture New Screenshots

To capture fresh screenshots:

```bash
cd frontend
npm run dev

# In another terminal:
node screenshot-basic.js
```

## Image Guidelines

- Resolution: 1920x1080
- Format: PNG
- Theme: Dark cinematic (Anti-AI Slop)
- Include: Full viewport with UI elements visible
