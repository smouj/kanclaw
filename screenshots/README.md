## Capturing More Screenshots

To capture additional screenshots:

```bash
cd frontend
npm run dev

# In another terminal, run Playwright:
node screenshot-capture.js
```

## Image Guidelines

- Resolution: 1920x1080
- Format: PNG
- Theme: Dark cinematic (Anti-AI Slop)
- Include: Full viewport with UI elements visible
