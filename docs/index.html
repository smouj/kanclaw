Quick Start

Get Started
in Seconds

Try the demo mode instantly. No setup required. Connect to OpenClaw gateway or run locally.

  • Demo mode available at /setup route
  • Connect to OpenClaw gateway for AI features
  • Keyboard shortcuts: Ctrl+K, Ctrl+1-5
  • i18n support: English, Spanish, French
  • Dark/Light theme toggle
Terminal
# Clone the repository
$ git clone https://github.com/smouj/kanclaw

# Install dependencies
$ npm install

# Start development server
$ npm run dev

✓ Ready on http://localhost:3000
