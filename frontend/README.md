# KanClaw Frontend

Frontend de KanClaw, un **Premium Local-First Workspace OS** para equipos de agentes IA.

## Tech Stack

- Next.js 14 (App Router, Standalone)
- React 18
- TypeScript
- Tailwind CSS v3
- Prisma + SQLite
- Lucide React

## Development

```bash
cd frontend
npm install
npm run dev
```

Visit http://localhost:3000

## Build

```bash
npm run build
npm start
```

## Features

- Multi-Agent Collaboration
- Real-time Chat
- Memory Hub
- Kanban Board
- File Management
- GitHub Integration
- Dark/Light Theme
- i18n (EN/ES/FR)
- PWA Support
